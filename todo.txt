Below is documentation information about the time of games:

Based on the search results, yes, the API provides date information for markets. The Market API includes date filters that indicate when markets start and end:

Date Fields (ISO 8601 format):

startDateMin / startDateMax - Filter markets by start date
endDateMin / endDateMax - Filter markets by end/expiration date
For sports markets specifically, there's a gameId field that links to the sports provider's game data.

You can filter markets by these dates using query parameters like:

GET /v1/markets?startDateMin=2025-01-01T00:00:00Z&endDateMax=2025-12-31T23:59:59Z
